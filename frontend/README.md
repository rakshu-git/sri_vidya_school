# Frontend - Temple Management Application

Frontend application for the Temple Management Application built with React, TypeScript, and Vite.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment variables:
   - Copy `.env.example` to `.env` (or use existing `.env`)
   - Fill in your Supabase and Razorpay credentials

3. Run development server:
```bash
npm run dev
```

## Production Deployment

Deploy to Vercel using the included `vercel.json` configuration.

See the main [deployment guide](../../brain/8be799f3-c879-43a4-8e08-df5a3cc39a07/deployment_guide.md) for detailed instructions.

## Features

- User authentication
- Donation management
- Transaction history
- Settings and preferences
- Multi-language support (English, Kannada)
- Responsive design

## Environment Variables

See `.env` file for required configuration.

## Build

```bash
npm run build
```

Output will be in the `dist` folder.
